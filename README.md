# TripBuddy 🌍✈️
### Yapay Zeka Destekli Akıllı Seyahat Asistanı

TripBuddy, kullanıcıların konumlarına veya seçtikleri şehirlere göre dinamik, yapay zeka destekli seyahat rotaları oluşturabilen kapsamlı bir Android mobil uygulamasıdır.

## 🚀 Özellikler
- **Yapay Zeka Rota Planlama:** Gemini AI entegrasyonu ile kişiselleştirilmiş seyahat ve gezi planları oluşturma.
- **Akıllı Önbellekleme (Caching):** API maliyetlerini ve yanıt sürelerini optimize eden iki aşamalı (Two-Tier) önbellekleme mimarisi.
- **Gelişmiş Harita Entegrasyonu:** Google Maps SDK ile rotaları görselleştirme ve konum tabanlı mekan önerileri.
- **Sosyal Ekosistem:** Firebase Cloud Messaging (FCM) ve SnapshotListeners ile gerçek zamanlı rota paylaşımı ve anlık bildirim altyapısı.
- **Çoklu Dil Desteği:** Jetpack Compose ile kurgulanmış, 5 farklı dil destekleyen modern ve minimalist kullanıcı arayüzü (UI).

## 🛠️ Kullanılan Teknolojiler & Kütüphaneler
- **Dil:** Kotlin
- **Arayüz (UI):** Jetpack Compose, Material Design 3
- **Mimari:** MVVM (Model-View-ViewModel), Clean Architecture prensipleri
- **Arka Uç & Veritabanı:** Firebase (Firestore, Authentication, Cloud Messaging)
- **Bulut Servisleri:** Google Cloud Platform (GCP)
- **Yapay Zeka:** Gemini AI API
- **Harita:** Google Maps SDK for Android
- **Bağlantı & Veri:** REST API, JSON, Coroutines & Flow

## 📌 Mimari Yapı
Projede kod okunabilirliği, test edilebilirlik ve sürdürülebilirlik amacıyla **MVVM** mimarisi kullanılmıştır. Kullanıcı deneyimini artırmak adına asenkron işlemler tamamen Kotlin Coroutines ve Flow ile yönetilmiştir.
